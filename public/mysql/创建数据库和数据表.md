我发现创建数据库和数据库表是两个不同的概念，哎，表示建立在数据库下面的

- 显示所有的数据库表
> 在登陆的前提下

```mysql
mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| qijian             |
| sys                |
| wordpress          |
| xiaqijian          |
+--------------------+
7 rows in set (0.01 sec)
```
- 显示某数据库下所有的表
> 前提是在某数据库，先要使用这个命令`use 数据库名称`
```mysql
mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| qijian             |
| sys                |
| wordpress          |
| xiaqijian          |
+--------------------+
7 rows in set (0.01 sec)
```
